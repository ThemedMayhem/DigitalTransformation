# DigitalTransformation

## Установка библиотек

### Python

Установка зависимостей для data science части

```
pip install -r libraries_for_ds.txt
```

### JS
Необходимо установить ```node.js```.

Установка зависимостей

```
cd client
npm install
```

Далее сборка проекта

```
npm run build 
```
